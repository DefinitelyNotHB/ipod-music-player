# iPod-Style Music Player (Web)
An interactive, browser-based recreation of the classic iPod music player experience.
Built entirely using vanilla HTML, CSS, and JavaScript, this project focuses on replicating a hardware-like user interface and interaction model inside the browser.

Designed to feel like a physical device. Implemented like real software.

# Features
  - Circular menu-style navigation inspired by the original iPod
  - Seamless audio playback with resume support across screens
  - Dynamic album art and song metadata rendering
  - Light / Dark mode toggle
  - Multi-language UI support
  - Fully client-side, zero frameworks, zero libraries

# Tech Stack
HTML – structure and layout
CSS – styling, themes, and UI polish
JavaScript – navigation logic, state handling, and audio control

# How to Run
Download the repository
Open the project folder
Double-click index.html

# Highlights
No frameworks or libraries used
Hardware-inspired UI logic
Clean, readable frontend code
